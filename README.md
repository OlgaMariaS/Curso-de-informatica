# Curso-de-informatica
Este site foi realizado para um trabalho acadêmico da disciplina de programação web.
Consiste na criação de uma página web que apresenta informações sobre um projeto para a comunidade, 
este projeto visa ensinar informática com foco no Google Worspace para jovens e adultos.

### As funcionalidades são: 
- ➕ Cadastro pessoal;
- ✏️ Edição do cadastro pessoal;
- 🔍 Login no site;
- 📋 Envio de feedbacks;
- Visualização das informações sobre o curso e UEM;

### O projeto consiste em 4 partes:
- Interface em HTML e CSS✅
- Integração com DOM ✅
- JavaScript, PHP e Laravel ✅
- Banco de dados em MySQL ✅
